# Dumpy

Easy file sharing straight from the command line

## Installation

Install it yourself using:

    $ gem install dumpy

## Usage

1. Run from terminal using the command:  

	`$ dumpy your_file.txt`

2. A download link will be copied to your clipboard. Just paste it.
